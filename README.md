### GetGank.io

<img src="/app/src/main/res/mipmap-hdpi/app_icon.png" width="200" height="200" />

GetGank.io是一个技术干货分享平台App，在工作日会每天更新一张妹子图、一个小视频和一些精选干货信息，包含Android、iOS等技术干货

**所有的数据均来自代码家的干货集中营:http://gank.io
